Das span ist ein Inline-Element, das im Textfluss angezeigt wird und standardmäßig nur so viel Platz einnimmt, wie sein Inhalt benötigt. Es wird häufig verwendet, um einen bestimmten Abschnitt von Text oder Inline-Inhalt zu markieren, auf den CSS-Styling oder JavaScript angewendet werden soll
Das div-Element ist ein Blockelement, was bedeutet, dass es standardmäßig den gesamten verfügbaren horizontalen Platz in Anspruch nimmt und auf einer neuen Zeile beginnt.
Es wird häufig verwendet, um Gruppen von Elementen zu erstellen und diesen Gruppen CSS-Stylin oder JavaScript-Funktionen zuzuweisen.
